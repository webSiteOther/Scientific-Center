نحن نوفر منهجاً تطبيقياً يركز على المشاريع العملية. سواء كنت طفلاً يبدأ بـ Scratch أو مهتماً بعلوم البيانات والذكاء الاصطناعي، فإن مساراتنا التعليمية مصممة لتنقلك خطوة بخطوة نحو سوق العمل أو الإبداع التقني.

الخطة الدراسية (Curriculum)

تفاصيل الكورسات مقسمة على 3 شهور (24 سيشن تدريبي)

كورس سكراتش (Scratch) للأطفال

مدخل ممتع لعالم المنطق البرمجي عبر السحب والإفلات.

الشهرالمحتوى التدريبي (8 Sessions)
الشهر الأولمقدمة عن الخوارزميات، واجهة البرنامج، وتحريك الكائنات (Motion & Looks).
الشهر الثانياستخدام الجمل الشرطية (If-statements) والتكرار (Loops) لعمل منطق اللعبة.
الشهر الثالثالمتغيرات (Variables)، البث (Broadcasting)، وإنشاء مشروع لعبة متكاملة.

كورس بايثون (Python Programming)

الشهرالمحتوى التدريبي (8 Sessions)
الشهر الأولأساسيات اللغة، Variables، Data Types، والـ Operators والتعامل مع المدخلات.
الشهر الثانيهياكل البيانات Data Structures (Lists, Dictionaries)، والـ Functions، والـ Control Flow.
الشهر الثالثمبادئ البرمجة كائنية التوجه OOP basics، والتعامل مع المكتبات، وعمل مشروع تطبيقي.

الذكاء الاصطناعي التطبيقي (Applied AI)

الشهرالمحتوى التدريبي (8 Sessions)
الشهر الأولمفاهيم الـ AI والـ Machine Learning، واستخدام أدوات مثل Teachable Machine.
الشهر الثانيأساسيات الرؤية الحاسوبية Computer Vision والتعرف على الأنماط والنصوص.
الشهر الثالثدمج نماذج الذكاء الاصطناعي في تطبيقات بسيطة AI Integration واستخدام APIs.

تصميم وتطوير المواقع (HTML, CSS, JS)

الشهرالمحتوى التدريبي (8 Sessions)
الشهر الأولإتقان HTML5 لبناء الهيكل التنظيمي للمواقع وكتابة أكواد نظيفة (Semantic HTML).
الشهر الثانيتنسيق المواقع باستخدام CSS3 والتعامل مع الألوان، الخطوط، ونظام Flexbox و Grid.
الشهر الثالثأساسيات JavaScript لإضافة التفاعلية للموقع والتعامل مع الـ DOM Manipulation.

تحليل البيانات (Excel & Power BI)

الشهرالمحتوى التدريبي (8 Sessions)
الشهر الأولAdvanced Excel (الدوال المتقدمة، Pivot Tables، تنظيف البيانات و Power Query).
الشهر الثانيمقدمة في Power BI Desktop، ربط مصادر البيانات المختلفة، وتجهيز الـ Data Model.
الشهر الثالثكتابة دوال DAX المتقدمة، وبناء Dashboards تفاعلية احترافية لعرض النتائج.

برمجة السي بلس بلس (C++)

الشهرالمحتوى التدريبي (8 Sessions)
الشهر الأولالـ Syntax الأساسي، المتغيرات، الإدخال والإخراج I/O، والعمليات الحسابية.
الشهر الثانيالجمل الشرطية Control Structures، الحلقات التكرارية Loops، والمصفوفات Arrays.
الشهر الثالثالدوال Functions، ومقدمة في الـ Pointers وتخصيص الذاكرة و Structs.
